Sheeps! - graphic design (pixelart)

Sheeps! is a mobile game dedicated to the phone Galaxy Note phone.

The application was designed to encourage the user to use the Galaxy Pen(special stylus for the Galaxy Note phone) and to test physics game engine.

The game tells the story of sheeps which go back at night from the pasture to the house. Be their shepherd who at every level must lead sheeps to the safe trailer in Your truck.

Technologies used: Illustrator, Photoshop, Graphics Gale.
